Call For Papers

Call for Papers! We're looking for speakers at WebClusters Conference 2012.
Talk should be planned for 35 to 45 minutes. Call for Papers will be closed at 22nd of April.

Conference topics:
High Performance clusters, real-time index, load balancing, full text search, cache policies, virtual server farms, frameworks for writing robust, distributed, highly available web applications (python, perl, ruby, java, js etc).

In application please include:
name, surname, e-mail of speaker, short professional bio (or linkedin/goldenline profile), talk topic,
short description.
Please send applications at address: info@webclusters.pl Note: Talks cannot contain product, company or solution presentation beyond what is essential for the talk. Marketing presentations are welcome in blocks for sponsors.
